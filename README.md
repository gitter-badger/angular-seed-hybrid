## Getting started

```bash
git clone --depth 1 https://github.com/jvitor83/angular-seed-hybrid
cd angular-seed-hybrid

# install the project's dependencies
npm install
```

### Web

```bash
# watches your files and uses livereload by default
npm start

# dev build
npm run build.dev
# prod build
npm run build.prod
```
